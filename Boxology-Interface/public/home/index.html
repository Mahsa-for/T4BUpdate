    How Tool4Boxology Works

    Tool4Boxology guides you through a structured workflow for creating, validating, and transforming Neuro-Symbolic AI architectures into knowledge graphs.

    1. Add Components or Patterns:
      Select single components or predefined elementary patterns from the left sidebar and place them onto the canvas.
    2. Connect & Define Clusters:
      Connect components to form Input → Process → Output triples. Select a complete triple and right-click to group it into a cluster.
    3. Name & Annotate:
      Assign meaningful names to components and clusters. Specify the appropriate semantic type for each element.
    4. Validate Your Architecture:
      Use the validation tool to ensure your boxology follows the required structural and semantic rules.
    5. Generate a Knowledge Graph:
      Click Create KG to automatically generate the corresponding RDF knowledge graph and store it in Virtuoso.
    6. Query & Explore:
      Run SPARQL queries directly from the interface to analyze, inspect, and explore your generated knowledge graph.
